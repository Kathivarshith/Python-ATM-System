# 🏧 Python ATM System

A simple Command Line Interface (CLI) ATM application built using Python. This project simulates basic ATM operations such as PIN authentication, balance inquiry, cash deposit, cash withdrawal, PIN change, and mini statement generation through a menu-driven interface.

This project was developed to practice Python programming fundamentals and understand how basic banking operations can be implemented using Python.

---

## 📌 Features

- 🔐 Secure PIN Authentication
- 💰 Check Account Balance
- 💵 Deposit Money
- 💸 Withdraw Money
- 🔄 Change ATM PIN
- 📄 View Mini Statement (Transaction History)
- 🖥️ Menu-Driven Command Line Interface

---


