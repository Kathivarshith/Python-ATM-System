# ATM-SYSTEM
A simple command-line ATM system built with Python that supports PIN authentication, balance inquiry, deposits, withdrawals, PIN updates, and mini statements.

# Python ATM System

## Overview

Python ATM System is a command-line application that simulates basic ATM operations. It allows users to securely access their account using a PIN and perform common banking transactions through a menu-driven interface.

This project was developed to strengthen my understanding of Python fundamentals, including functions, loops, conditional statements, lists, and user interaction.

## Features

- PIN Authentication
- Balance Inquiry
- Deposit Money
- Withdraw Money
- Change PIN
- Mini Statement
- Menu-Driven Interface

## Technologies Used

- Python 3

## Concepts Used

- Functions
- Loops
- Conditional Statements
- Lists
- Global Variables
- User Input
